# Next.js Authentication Pages (Signup & Signin)

This project is a simple Next.js application demonstrating **file-based routing** using the App Router.  
It includes basic **Signup** and **Signin** pages built with **React, Next.js, and Tailwind CSS**.

##  Features
- File-based routing using `app/` directory
- Signup page (`/signup`)
- Signin page (`/signin`)
- Server-side rendering (SSR) with Next.js
- Styled using Tailwind CSS
- Clean and beginner-friendly structure

##  Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS

##  Folder Structure
app/
├─ page.tsx → Home page
├─ signup/
│ └─ page.tsx → Signup page
├─ signin/
│ └─ page.tsx → Signin page
